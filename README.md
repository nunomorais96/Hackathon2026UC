# AlphaLens

AlphaLens is a multi-agent AI investment research copilot.

It helps retail investors transform public financial data into an explainable investment research brief.

## Problem

Retail investors often make decisions based on fragmented information, market hype, and incomplete financial understanding.

AlphaLens addresses this by using multiple AI agents to analyze public financial metrics, risk indicators, sentiment drivers, and portfolio fit.

## Input

The user provides:

- Stock tickers
- Risk profile
- Investment horizon

Example:

```text
NVDA, AMD, PLTR, AMZN
Moderate-Aggressive
5 years
